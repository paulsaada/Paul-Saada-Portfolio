Tap Champion Game

HTML 5, CSS 3, JavaScript

This used to be a very simple tutorial project from Colt Steele's Web Developer Bootcamp course. I revamped the UI and turned the mouse click controls into keyboard controls so two players could play. So grab a friend, agree on a score, use the Z and M keys on your keyboard to play, and the first player to reach the score wins the round!
